== Description ==

Konec nahrávání české mo souboru do BuddyPress. Již nepotřebujete žádné FTP, nepotřebujete k tomu žádné znalosti ani webmastera. 
Stačí jen aktivovat plugin ve vaši administraci a překlad je hotov.

Od verze 2.4.x budou přídány také další pluginy pro BuddyPress vše v jednom. 

== Installation ==

1. Nainstalujte plugin přes administraci v Pluginy.
1. Activujte plugin.
