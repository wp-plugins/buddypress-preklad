== Description ==

Plugin přidá další jazykové překlady pro Váš BuddyPress.

Momentálně je k dispozici :

* Čeština
* Slovenčina
* Croatian
* Dutch
* Italian
* Japanese
* Korean
* Norwegian (Bokmål)
* Portuguese (Brazil)
* Portuguese (Portugal)
* Russian
* Spanish (Spain)
* Thai
* German 

== Installation ==

Instalace je velmi jednoduchá

1. Nainstalujte plugin přes administraci v Pluginy.
1. Activujte plugin.
